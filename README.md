# ollama-chatbot
LLM-powered chatbot using Ollama and Streamlit, running completely on local hardware.

A local AI chatbot built using Ollama and Streamlit. The application provides an interactive chat interface and runs Large Language Models (LLMs) locally without relying on external APIs.
- Interactive chat interface
- Local LLM inference using Ollama
- Real-time responses
- Simple and user-friendly UI
- No API costs
- Python
- Streamlit
- Ollama
- Llama 3
- Launch the application.
- Enter your question in the chat box.
- Receive responses generated by the local LLM.
